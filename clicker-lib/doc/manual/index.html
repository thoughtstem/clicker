 (require clicker-lib/start) package: clicker-lib

Provides a generic start function for other healer games to build upon.

syntax

(start avatar (food ...) (friends ...) (enemies ...))

The basic starter for a healing game. It is a macro because we want people to type:

(start cat (dog apple))

... not:

(start cat (list dog apple))

So we capture the user’s code, package the parameters into lists and call whatever function has been "bound" with bind-start-to. This allows different variations on the healing game to be congifured by defining a function (e.g. my-start) and binding it:

(bind-start-to my-function)

The function should be one that takes between 0 and 4 parameters. In most cases, it will directly call generic-start, which does the heavy lifting for the game.
